Beauty Salon Management

Master Management

Master Schedule

  • Track master availability
  • Manage specializations (Hairdresser, Manicure, Pedicure, Beautician, Masseur)
  • View workload reports
  • Real-time conflict detection

Client Management

  • Client profiles with personal information
  • Loyalty program with tiered discounts
  • Appointment history tracking
  • Birthday tracking and reminders
  • Visit statistics

Appointment System

  • Online booking system
  • Conflict detection (double bookings)
  • Status tracking (Planning, Finished, Cancelled)
  • Automated reminder system
  • Service history

Service Catalog

  • Service management with pricing
  • Duration tracking
  • Service history
  • Category management

Reminder System

  • Automated reminders for recurring services
  • Custom notes
  • Status tracking
  • Email notifications

Reports

  • Master workload reports
  • Service statistics
  • Client visit history
  • Revenue analysis
  • Performance metrics

Security Features

  • Two user groups: Beauty Salon User and Admin
  • Role-based access control
  • Data protection
  • Audit trail

Installation

  1. Install the module through Odoo's Apps menu
  2. Go to Settings > Users & Companies > Groups
  3. Assign users to appropriate groups
  4. Configure system settings
